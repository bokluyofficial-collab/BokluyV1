Bokluy Refund Policy Digital service
Important: Top-ups and digital services are usually non-refundable once delivered. Refund decisions depend on verification and the specific case.

1. Eligible Refund Cases

  • Payment confirmed but service not delivered due to a verified system issue.
  • Duplicate payment for the same order (verified by the owner).
  • Technical error caused by Bokluy that prevents delivery.

2. Non-Refundable Cases

  • Incorrect Player ID / Server ID / game account details entered by the customer.
  • Order already delivered / top-up completed successfully.
  • Refund requests after a long delay when verification is not possible.
  • Fraud, chargeback abuse, or suspicious activity.

3. Verification & Processing

  • Refunds are only considered after the owner verifies payment records.
  • Processing time depends on the payment method and investigation.
  • We may request screenshots or transaction details to confirm the issue.

4. How to Request a Refund

Contact support via the links on the site (Telegram / Social channels) and include:

  • Order details (game, package, amount)
  • Transaction reference / proof of payment
  • Your Player ID / Server ID
  • Description of the issue

5. Changes

We may update this policy at any time. Continued use of Bokluy means you accept the updated policy.
